    Conocimientos

    Html

    Avanzado

    HTML es un lenguaje de marcado utilizado para crear páginas web.Se utiliza para crear la estructura básica de una página web, incluyendo los títulos, los párrafos y otros elementos que conforman el contenido de la página.

    Css

    Avanzando

    CSS es el acrónimo de "Cascading Style Sheets" o "Hojas de Estilo en Cascada" en español. Es un lenguaje de estilo que se utiliza para controlar la apariencia y presentación de una página web. Con CSS, es posible definir el diseño, el color, la fuente, el tamaño y otros aspectos visuales de una página web.

    Javascript

    Intermedio

    JavaScript es un lenguaje de programación de alto nivel que se utiliza principalmente en la web. Es un lenguaje de programación interpretado que se ejecuta en el navegador web del usuario y permite agregar interactividad y dinamismo a las páginas web.

    Php

    Intermedio

    PHP es un lenguaje de programación utilizado principalmente para el desarrollo de aplicaciones web dinámicas y sitios web. Es un lenguaje de programación interpretado que se ejecuta en el servidor web y se utiliza para procesar datos del lado del servidor.

    Mysql

    Intermedio

    MySQL es un sistema de gestión de bases de datos relacionales (RDBMS) que se utiliza para almacenar, organizar y gestionar datos. Es uno de los sistemas de gestión de bases de datos más populares y se utiliza ampliamente en el desarrollo de aplicaciones web.

    Bootstrap

    Intermedio

    Bootstrap es un framework de diseño web que se utiliza para desarrollar sitios web y aplicaciones web responsivas y móviles. Es uno de los frameworks más populares para el diseño web y es utilizado por muchos desarrolladores web para crear sitios web y aplicaciones web modernas y atractivas.

    tailwindcss

    Basico

    Tailwind CSS es un framework de diseño de interfaz de usuario (UI) de código abierto que utiliza clases de CSS predefinidas para construir interfaces de usuario personalizadas. Es una herramienta muy útil para desarrolladores que buscan una forma rápida y eficiente de construir interfaces de usuario limpias y atractivas.

    Laravel

    Intermedio

    Laravel es un framework de desarrollo de aplicaciones web PHP de código abierto y muy popular.Laravel está diseñado para ser fácil de usar y para ayudar a los desarrolladores a construir aplicaciones web rápidamente. Proporciona herramientas y características que simplifican la tarea de desarrollar aplicaciones web, como un sistema de enrutamiento elegante, controladores, vistas, migraciones de bases de datos y autenticación de usuarios.

    Figma

    Intermedio

    Figma es una herramienta de diseño gráfico y prototipado de interfaces de usuario que se ejecuta completamente en línea. Es una plataforma colaborativa que permite a los diseñadores y desarrolladores trabajar juntos en tiempo real en proyectos de diseño y prototipado de aplicaciones web y móviles.

    Xd

    Intermedio

    Adobe XD es una herramienta de diseño de experiencia de usuario (UX) y diseño de interfaz de usuario (UI) desarrollada por Adobe Systems. Es una plataforma que permite a los diseñadores crear y compartir prototipos y diseños interactivos para sitios web, aplicaciones móviles y otras plataformas digitales.

    Git

    Basico

    Git es un sistema de control de versiones de código abierto que se utiliza para rastrear los cambios en el código fuente de un proyecto de software.
